# DPCATCH Homepgae



The website is built with quarto.


## Build locally

To build the homepage, quarto needs to be installed.


### Live preview

        run::

        quarto preview

This opens the rendered homepate.


### Render locally

        run::

        quarto render

This generates the html code, located at public/index.html.

The index.html can then be opened with a web browser.

Push the changes to Github, and the Homepage will be visible at https://dpcatch.github.io/



## Build oline

Update files in the repository directly.

Github will then render the homepage directly through a Github action.

The homepage is visible at https://dpcatch.github.io/
